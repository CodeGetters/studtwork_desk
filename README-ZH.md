## Install

需要:

- node >= `16`

```shell
$ git clone https://github.com/CodeGetters/studTWork.git

# 推荐
$ yarn install

# or using npm

$ npm install
```

## 使用

```shell
# dev
$ yarn dev

# build
$ yarn build
```
