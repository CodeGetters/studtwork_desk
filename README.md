## Install

require:

- node >= `16`

```shell
$ git clone https://github.com/CodeGetters/studTWork.git

# Recommend to use yarn
$ yarn install

# or using npm

$ npm install
```

## Usage

```shell
# dev
$ yarn dev

# build
$ yarn build
```
