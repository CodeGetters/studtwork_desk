## Install

需要:

- node >= `16`

```shell
$ git clone https://github.com/CodeGetters/studTWork.git

# 推荐
$ yarn install

# or using npm

$ npm install
```

## UnoCSS

查看 UnoCSS 编译后的文件---运行后查看 http://127.0.0.1:9000/__unocss#/

## 使用

```shell
# dev
$ yarn dev

# build
$ yarn build
```
